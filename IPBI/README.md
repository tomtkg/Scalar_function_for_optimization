# Inverted PBI approach        IPBI approach

このスカラー化関数の等高線は重みベクトルで曲がります．  
パラメータ*θ*の値によって等高線が曲がる角度が変わります．  
*θ*=0のとき等高線は曲がらずWSと同じ動きになります.  
*θ*=1のとき等高線は直角に曲がります.  
PBIがIdeal pointを使うのに対してIPBIはreference pointを使います
